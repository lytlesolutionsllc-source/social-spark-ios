# Social Spark iOS

Creator-powered missions for brands that want real social momentum.

## MVP Features

- Welcome + account type picker
- Creator dashboard
- Available missions + mission details
- Spark Run game mode
- Proof submission
- Spark Wallet
- Brand dashboard
- Create campaign flow
- Package selection
- Campaign status
- Admin dashboard
- Admin settings
- Mock data only (no external integrations)

## Getting Started

1. Open `SocialSpark.xcodeproj` in Xcode from the repository root (`social-spark-ios/SocialSpark.xcodeproj`)
2. Select the `SocialSpark` scheme
3. Run on iOS 16+ simulator or device

# Social Spark iOS

Creator-powered missions for brands that want real social momentum.

## MVP Features

- Creator and Brand account types
- Spark Run game mode with coin collection
- Mission dashboard
- Proof submission
- Spark Wallet
- Admin settings
- Mock data for demonstration

## Getting Started

1. Open `SocialSpark.xcodeproj` in Xcode
2. Select iOS 15+ as target
3. Run on simulator or device
